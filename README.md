# Midnight Zombie Convenience

## Project Description
A comic about a middle-aged man surviving in a convenience store after a zombie apocalypse.

## Created:
2026-03-19

## Artist/Author:

## Style:
1990s noir anime / mature seinen aesthetic
- Gritty, atmospheric lighting
- Realistic rugged character design
- Focus on mood and survival tension
